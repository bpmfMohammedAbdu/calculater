# 🧮 JavaScript Calculator

A basic calculator built with HTML, CSS, and JavaScript.

## ✨ Features
- Perform +, −, ×, ÷, %
- AC and DEL buttons
- Clean and responsive design

## 🛠 Built With
- HTML
- CSS
- JavaScript

## 📂 Files
- `index.html` – Page structure
- `style.css` – Styling
- `script.js` – Logic
- `package.json` – Project info
